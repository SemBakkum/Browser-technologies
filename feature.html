Core functionality Funda app

De core functionaliteit van de Funda app is het gebruik van geolocatie. Zonder deze functionaliteit is de app eigenlijk niet te gebruiken en ook niet nuttig. Door dat de gebruiker niets hoeft in te voeren en de gps van zijn/haar device gebruikt wordt is er geen mogelijkheid om het zelf in te voeren. Mocht het apparaat geen gps ondersteunen dan zou er als fallback een input field moeten komen waar de gebruiker de locatie kan invoeren. In de huidige code wordt er al gekeken of het apparaat geolocatie ondersteunt als dat zo is wordt de huidige code uitgevoerd. Als fallback kan er een else statement aan toegevoegd worden die de input box displayed. Of nog beter de input box wordt standaard gedisplayed en als geolocatie wordt ondersteund dan wordt de input box weg gehaald.
